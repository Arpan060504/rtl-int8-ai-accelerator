RTL INT8 AI Accelerator
Overview

This repository documents the incremental development of an INT8 AI accelerator in Verilog HDL.

Instead of directly implementing a complete accelerator, the project builds each architectural component from scratch to understand the design trade-offs used in modern AI hardware such as Google TPU, NVIDIA Tensor Cores, and Apple Neural Engine.

The project progresses through multiple versions, where every version introduces a new architectural concept.
